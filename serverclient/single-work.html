Server Client Dateitransfert

Das Projeckt wurde in Java,javafx und css entwickelt. Dieses Proogramm zeigt erstmal eine einfache Verbindung zwischen einem Server und einem Client. Dazu wie das Datentransfert mittels TCP/IP-Verbindung über Sockets realisiert. Der Client hat viele Möglichkeiten. (wie zum Beispeil Dateien in Server hochzuladen, herunterladen, löschen und anzeigen) Dafür sollte der Server ein ACK ertmal schicken.
